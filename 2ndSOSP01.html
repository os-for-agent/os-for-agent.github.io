The Challenge

The legacy OS is holding AI back.

AI agents have outgrown their sandboxes. Today's autonomous agents plan, invoke tools, and collaborate continuously. Yet, they are bottlenecked by decades-old OS abstractions—processes, threads, and sockets—that were never built for dynamic, semantically rich AI workloads.

To unleash AI at scale, the OS itself must become agentic. We must rebuild from the kernel up: rethinking isolation, semantic scheduling, and context management.

Core Themes & Topics

  • New Abstractions: Process, container, and multikernel enhancements tailored for agent execution.
  • Dynamic Sandboxing: Lightweight, iron-clad runtimes for executing unverified, agent-generated code.
  • Semantic Scheduling: Resource management aware of multi-agent and LLM prompt workloads.
  • Episodic Memory: Long-lived state abstractions managing context and prompt histories natively.
  • Agent Observability: Deep provenance tracking and debugging for autonomous operations.
  • Compiler–OS Co-design: Adaptive, agent-aware JIT execution strategies.
  • Self-Managing Systems: Agents taking the wheel for kernel tuning, anomaly detection, and failure recovery.
